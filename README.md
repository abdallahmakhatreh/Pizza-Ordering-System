# Pizza Ordering System

A simple and interactive desktop application built in C# using Microsoft Visual Studio and Windows Forms. This project demonstrates basic graphical user interface (GUI) logic, event-driven programming, and dynamic calculations based on user input.

---

## 🛠️ Tech Stack & Tools
* **Language:** C#
* **Framework:** .NET Framework (Windows Forms)
* **Development Environment:** Microsoft Visual Studio

---

## 🍕 Features & Functionality
* **Dynamic Price Calculation:** Automatically updates the total price in real-time as the user selects different pizza sizes, crust types, and toppings.
* **Component State Tracking:** Uses RadioButtons for mutually exclusive choices (Size and Crust) and CheckBoxes for multiple selection choices (Toppings).
* **Order Summary:** Displays a text-based breakdown of the selected items before finalizing the order.
* **Form Reset:** A built-in reset action button that instantly restores all options to their default states and clears the pricing total.
